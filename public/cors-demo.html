Percy Maestro — cross-origin iframe serialization demo

This page is served from localhost:8000. Each iframe below points to a different external origin. When Percy's SDK captures this page, its capture server uses CDP Target.getTargets + Target.attachToTarget to enumerate the iframe execution contexts, injects @percy/dom into each, serializes the iframe DOM, and attaches the result under domSnapshot.corsIframes — identical behavior to @percy/playwright.

Iframe 1 — httpbin.org/html

Sample HTML page; different origin from localhost

Iframe 2 — jsonplaceholder.typicode.com

JSON placeholder API home; another different origin
